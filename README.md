# Resume Builder - Full Stack Application

## 🎯 Overview
A professional resume builder that allows interns to create, 
customize, and download resumes as PDFs.

## ✨ Features
- Dynamic resume templates
- Real-time preview
- PDF generation with PDFKit
- Cloudinary image upload
- MongoDB database
- Responsive design

## 🛠️ Tech Stack
- **Frontend:** React 18
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **PDF:** PDFKit
- **Storage:** Cloudinary

 ## 🚀 How to Run Locally

### Backend:
```bash
cd resume-builder-backend
npm install
npm run dev
```

### Frontend:
```bash
cd resume-builder-frontend
npm install
npm start
```
